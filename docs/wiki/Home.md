## Getting Started
- Prerequisites: Node.js (LTS recommended), npm.
- Clone: `git clone https://github.com/talitalima-tech/drip-store.git`
- Install: `npm install`
- Development: `npm run dev` then open http://localhost:5173/
- Build: `npm run build`
- Preview build: `npm run preview`
